## Decisions
### legacy-peer-deps workaround
**Choice:** Added .npmrc with legacy-peer-deps=true at project level
**Rationale:** Pre-existing peer dep conflict between zod v4 and @tanstack/zod-adapter (requires zod ^3). The lockfile already resolves this correctly; legacy-peer-deps prevents npm from blocking installs due to the mismatch. This is the standard workaround until @tanstack/zod-adapter adds zod v4 peer support.
