## 2. Routing

- **Library**: TanStack Router v1.161.3
- **Mode**: File-based routing via `@tanstack/router-plugin/vite` with `autoCodeSplitting: true`
- **Route definition**: Each file exports `Route = createFileRoute('/path')({ component, validateSearch? })`
- **Search params**: Zod schemas with `zodValidator()` from `@tanstack/zod-adapter`
- **Auto-gen**: `routeTree.gen.ts` regenerated on file changes (excluded from lint/TS checks)
- **Type registration**: `router.ts` declares module augmentation for `Register.router`

## 6. API Client & Data Fetching

### API Client
- `apiClient<T>(path, options?)` in `api/client.ts`
- Prepends `/api` to all paths
- Throws typed `ApiError(status, detail)` on non-OK responses
- Vite dev server proxies `/api` -> `http://localhost:8642`

### TanStack Query
- Version: 5.90.21
- Global config: `staleTime: 30_000`, `retry: 2`, `refetchOnWindowFocus: false`
- Query key factory: `queryKeys` object with hierarchical keys per domain
- Pattern: hooks wrap `useQuery()` with typed `queryKey` + `queryFn`
- Dynamic staleTime: terminal runs get `Infinity`, active runs get shorter stale + poll

## 7. Component Patterns & Conventions

### Function style
- Named function declarations (not arrow const)
- `export function ComponentName({ prop1, prop2 }: ComponentNameProps) { ... }`

### Page structure
```tsx
export const Route = createFileRoute('/path')({
  validateSearch: zodValidator(schema),  // optional
  component: PageComponent,
})

function PageComponent() {
  // hooks at top
  // return JSX
}
```

### Layout pattern
```tsx
<div className="flex h-full flex-col gap-4 p-6">
  <h1 className="text-2xl font-semibold text-card-foreground">Title</h1>
  <div className="flex min-h-0 flex-1 gap-4">
    {/* Left panel */}
    <div className="flex w-80 shrink-0 flex-col overflow-hidden rounded-xl border">...</div>
    {/* Right panel */}
    <div className="flex-1 overflow-auto rounded-xl border">...</div>
  </div>
</div>
```

### State patterns
- Loading: skeleton divs with `animate-pulse bg-muted` + fixed width
- Error: centered `text-destructive` message
- Empty: centered `text-muted-foreground` message
- Data guard: `if (isLoading) return <Skeleton />`; `if (isError || !data) return <Error />`

### Imports
- Path alias: `@/` maps to `src/`
- UI components from `@/components/ui/...`
- Domain components from `@/components/{domain}/...`
- API hooks from `@/api/{domain}`
- Types from `@/api/types`
- Utilities from `@/lib/utils`
