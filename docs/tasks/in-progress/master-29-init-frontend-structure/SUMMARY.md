## Issues Encountered

### Broken favicon reference in index.html
The scaffold's `index.html` referenced `/vite.svg` via `<link rel="icon">` but `public/vite.svg` was deleted in step 1. This caused a console 404 in dev mode. The page `<title>` also retained the scaffold default `frontend`.

**Resolution:** Removed the `<link rel="icon">` tag entirely and changed the title to `llm-pipeline`. Fixed in commit `5e94178`.

### Hatchling exclude list missing index.html, README.md, .gitignore
The `[tool.hatch.build.targets.wheel]` exclude list (from PLAN.md) did not cover `index.html`, `README.md`, or `.gitignore`. These non-dist dev files would be bundled into the wheel, adding minor payload.

**Resolution:** Added all three paths to the exclude list in `pyproject.toml`. Fixed in commit `6311c5c`.
