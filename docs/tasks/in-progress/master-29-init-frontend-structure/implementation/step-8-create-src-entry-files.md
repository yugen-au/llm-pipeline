## Decisions
### QueryClient defaults
**Choice:** staleTime: 30s, retry: 2, refetchOnWindowFocus: false
**Rationale:** 30s stale time reduces unnecessary refetches for pipeline data that doesn't change rapidly. 2 retries handles transient failures. Disabled window focus refetch avoids unexpected re-requests during dev.

### Dark mode application method
**Choice:** `document.documentElement.classList.add('dark')` in main.tsx
**Rationale:** Runs before React render, ensures dark class is present for initial paint. Avoids FOUC. Can be replaced with theme persistence logic in task 42.

### Router codegen method
**Choice:** `npx @tanstack/router-cli generate` instead of `npm run dev`
**Rationale:** Cleaner than starting/stopping dev server. Produces identical routeTree.gen.ts without requiring port availability.
